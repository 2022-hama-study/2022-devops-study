# 4. 컨테이너를 실행해 보자

## 1. 도커 엔진 = 도커 데몬
### 사전 작업
docker 명령어를 sudo 없이 사용하면 Permission 오류가 뜰것이다. 아래 명령어로 해결 할 수 있다.
```bash
sudo usermod -aG docker $USER
```

### 도커 엔진 시작
```bash
sudo systemctl start docker
```

### 도커 엔진 종료
```bash
sudo systemctl stop docker
```

### 도커 엔진 자동 실행 설정
```bash
sudo systemctl enable docker
```
